Mali Thai Bistro

Mali Thai Bistro is a family-owned traditional Thai restaurant founded in 2015, serving authentic Thai cuisine in Lee's Summit, MO. Even though they earned popularity on Facebook, TripAdvisor, and Yelp reviews, the restaurant did not have an official website. Oftentimes, customers could not figure out the menu list online before they actually arrive in-person at the restaurant.

This Single Page Application (SPA) is "mobile-first" designed i.e.,
a collapsible navbar and a responsive menu section making this web easy-to-read on small screens. In order to meet the user's expectation when they check the restaurant's website, this web app integrates the restaurant's location from Google Maps, news and events update from Facebook page, and a floating 'scroll to top' arrow-up button for easy navigation.

What's For Dinner?

A meal plan generator browser-based app that allows the user to save searched recipes to weekly meal planner. This app is developed because busy parents often run out of ideas what to feed their children for dinner, and other caregivers have no clue what the meal plan is. In addition, keywords about kid-friendly dinner ideas and meal planning are frequently searched for in google.

I designed modern-looking UI by implementing Bulma, Google Web Fonts, Font Awesome, and “modal” like alerts. I wrote JavaScript code that allow the user to add/edit/remove data on the weekly planner template, and register their login and save recipes to Firebase database.

Our team used Github with 120+ commits in 1.5 weeks along with Slack and Trello for communication and project management. We applied an MVC architecture by creating separate Javascript file for different tasks. This helped separate concerns, organize the codebase, and streamline the communication between all parties.
